## Key design decisions (pinned)

1. **"Allow all" sentinel for model/killzone sets**: `PostLLMParams` uses
   frozenset semantics where an **empty frozenset = reject everything**
   (testable). "Allow all" is the module-level sentinel `ALL = frozenset({"*"})`
   — rescore treats `{"*"}` as "no filter". `default_post_params()` uses `ALL`
   for both (baseline = no filtering = today's behavior).

2. **R:R recompute in rescore**: reuses the EXACT formula from `validate_rr`
   via a shared `compute_rr` helper (the only permitted trading-layer change is
   config threading, so the helper lives in rescore, not alert.py). A
   `test_rr_recompute_matches_validate_rr` pins the agreement. A degenerate
   (risk==0) taken trade is downgraded to no_trade (matches validate_rr's
   degenerate_stop branch).

3. **Objective = PerformanceReport.expectancy**: the single source of truth is
   `PerformanceReport._overall_metrics(traces)["expectancy"]` = mean(actual_rr)
   over all traces with no_trade/no_fill/expired = 0. REUSED (never recomputed
   independently). `taken_trades` = filled_count on the re-scored set.

4. **MIN_TRADES guard**: `evaluate` returns `(-inf, metrics)` when
   `taken_trades < min_trades`. `metrics` always populated. `-inf` strictly
   below; finite at exactly `min_trades`.

5. **Walk-forward folds**: rolling, half-open `[start, start+train_days)` /
   `[start+train_days, start+train_days+test_days)`. **Per-fold disjointness
   asserted** (`_assert_disjoint`, release blocker). step_days advances the
   window. Folds with empty train or test dropped. Timestamps parsed once to
   aware datetimes (naive → ET, matching report.py); fold bounds in UTC.

6. **Tie-break in search**: first-in-grid-order wins (deterministic). Documented.

7. **All-below-min_trades fallback**: returns the baseline (default) params with
   their score so the fold still records a comparable result. Documented.

8. **Aggregation in report**: reports BOTH mean-of-folds AND trade-weighted
   (pooled) for OOS expectancy, so small-sample folds can't hide. Documented.

9. **Overfitting warning**: fires when (mean IS − mean OOS) > 0.5R threshold
   OR tuned OOS ≤ baseline OOS. The latter emits "Tuning added nothing; use
   baseline."

10. **Tier-2 recording**: cost = `n_configs × total_fires × $0.003` (one
    full-span backtest per config; folds do NOT multiply cost). Per-config
    `runs/{config_hash}/` dirs. Cost gate mirrors Phase 3A spend gate.
    Idempotent resume via BacktestEngine.
