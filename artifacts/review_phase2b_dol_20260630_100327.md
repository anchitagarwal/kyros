### c. ERL/IRL classification — PASS
FVG midpoint inside the dealing range → scope `internal`, role `fvg_ce`;
swing/equal levels at/above boundary → `external`. Scope refinement uses
`range_low < level < range_high` (strictly inside → internal; at/beyond
boundary → external), overriding the source default. Boundary-inclusivity
verified at 100/200/199.99. Tests pin both inside and at-boundary cases.

### d. Cycle-state derivation — PASS
- BSL sweep, no reversal → last_swept_erl_side `buyside`, target `sellside`,
  current_leg `seek_irl`, next_draw `irl`. Verified.
- Reversal BOS/displacement strictly after sweep → `expand_to_erl` /
  next_draw `erl`. Verified.
- No sweep → `None`. Verified.
- Stateless: same input twice → equal output; no cross-snapshot memory
  (derived fresh from snapshot-local sweeps/structure/displacements).

### e. Cycle-aware DOL AUGMENTS, never replaces — PASS
- cycle None or `dol_use_cycle=False` → identical object to `nearest_dol`.
- seek_irl → nearest unswept INTERNAL pool in reversal direction (verified
  internal/ote selected when one exists in the reversal direction).
- expand_to_erl → nearest unswept EXTERNAL pool on target side beyond
  equilibrium.
- Bias conflict → returns `_nearest_dol` and flips the stored
  `agrees_with_htf_bias` to False (mutation visible to the LLM).
- Falls back to `_nearest_dol` when cycle selection finds nothing.
`_nearest_dol` never altered.
